Customer Data Management
=========================

## Used Frameworks
- Intellij JavaFX
- Maven


## Brain build script:

### Dependencies:

#### 1. Maven 3:
- Ubuntu: sudo apt-get install maven
- macOS: brew install maven

#### 2. Java 7 || 8:
- Ubuntu: sudo apt-get install openjdk-8-jdk
- macOS: brew cask install java

### Run from terminal:
- ./build.sh
